# Interview Patterns for Google & Microsoft

A concise, practical guide to interview patterns and error handling for Google and Microsoft, maintained by mwakidenis.

## Features
- HashMap and Dictionary concepts (C++ & Python)
- Error handling best practices
- Common interview patterns (Two Sum, Anagrams, Frequency Counting, Sliding Window, Mapping)
- Visual diagrams and step-by-step explanations
- Company-specific expectations and follow-up questions

## Structure
- `/docs` — Main documentation
- `/docs/problems` — Practice problems

- Company-specific expectations (Google, Microsoft)

### 2. Practice Problems
- Two Sum
- Anagrams
- Frequency Counter
- Sliding Window
- Key-Value Mapping

### 3. Visual Aids
- Flowcharts for algorithm steps

### 4. Code Examples

- Side-by-side syntax and structure comparisons

### 5. Interview Preparation
- Common pitfalls and best practices
- Follow-up questions and optimization strategies

- `/docs/index.md`: Entry point and table of contents

## Usage
1. Clone the repository:
   ```bash
   git clone https://github.com/mwakidenis/Interview-Patterns-for-Google-Microsoft.git
   ```
2. Explore the `/docs` folder for detailed guides and examples.
3. Use the practice problems to test your understanding.
4. Review visual diagrams for quick reference.
